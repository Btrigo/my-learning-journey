> Background
I love how infrastructure works — not just at the surface level. The fact that there are fiber cables running across the floor of the ocean carrying signals that get translated into readable information in milliseconds. That when you load a webpage, you're probably hitting a CDN edge node, not the origin server. That you can define an entire network topology in code and a cloud provider will go build it for you. That stuff fascinates me. My background is physical; I've racked servers, terminated fiber, managed structured cabling, and handled remote hands operations in a live colocation environment. Now I'm building upward into cloud architecture and infrastructure engineering.
