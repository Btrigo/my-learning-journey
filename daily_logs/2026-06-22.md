### Key Takeaways
- React component = function returning JSX; `useState` is how it "remembers" values across re-renders
- Arrays = ordered lists accessed by position; objects = labeled collections accessed by key — almost all nested data is just these two stacked
- "Apex/root domain" refers to a domain's own bare name (no subdomain prefix) — unrelated to the global DNS root server hierarchy, just an unfortunate shared word
- CNAME can't coexist with other required record types (NS/SOA) at a name, which is why it's banned at the apex — Route 53's Alias record exists specifically to work around that
- OAC and OIDC are both federated/delegated trust patterns, but authenticate entirely different relationships (service-to-service vs. CI/CD-to-cloud)
