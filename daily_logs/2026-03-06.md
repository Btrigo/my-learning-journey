## 🧠 What I Learned / Key Takeaways
- **Amazon CloudWatch** collects metrics, logs, and events from AWS resources and can trigger alarms or automated actions.
- **AWS CloudTrail** records API activity and tracks who performed actions, when they occurred, and which resources were affected.
- **AWS Config** evaluates and tracks configuration changes to AWS resources and helps enforce compliance policies.
- **AWS Trusted Advisor** provides recommendations to improve cost optimization, performance, security, and fault tolerance.
- **AWS Artifact** provides on-demand access to AWS security and compliance reports and agreements.
- **AWS Organizations** allows centralized management of multiple AWS accounts, including billing consolidation and policy enforcement.
- **Service Control Policies (SCPs)** are used in AWS Organizations to set permission guardrails across accounts.
- **AWS License Manager** helps manage software licenses and enforce license usage limits across AWS resources.
- **AWS Service Catalog** enables organizations to provide curated, approved AWS resource templates for self-service deployment.
- **AWS Control Tower landing zones** provide a preconfigured multi-account environment aligned with security and compliance best practices.
