## Key Takeaways

- **Elastic IP billing (updated Feb 2024):** All EIPs are charged regardless of state — in use, idle, or associated with a stopped instance. The old rule (free when attached to running instance) no longer applies.
- **VPC Gateway Endpoints:** S3 and DynamoDB only, free, traffic stays on AWS private network — no internet gateway or NAT needed. Interface Endpoints cover everything else and are paid.
- **Security Groups:** Allow rules only (no explicit deny). Implicit deny on all inbound by default. All outbound allowed by default. Stateful — return traffic automatically allowed.
- **NACLs:** Both allow AND deny rules. Stateless — checks every packet every time including return traffic. Default NACL allows all; custom NACL denies all until rules added.
- **On-Demand vs Auto Scaling:** On-Demand is a pricing model. Auto Scaling is a capacity feature. Don't conflate them on exam questions about pricing.
- **Compute Savings Plans vs Reserved Instances:** Reserved Instances do NOT cover Lambda. Compute Savings Plans cover EC2, Fargate, AND Lambda.
- **Support Plans — TAM distinction:** Enterprise On-Ramp = pool of TAMs. Enterprise = designated TAM. keyword "designated" = Enterprise.
- **Full Trusted Advisor:** Business plan and above only.
- **Pricing Calculator vs Cost Explorer:** Calculator = pre-deployment estimate. Explorer = historical analysis and forecasting.
- **AWS Budgets vs Cost Explorer:** Budgets = proactive alerts before overspending. Explorer = reactive analysis of past spend.
- **S3 Intelligent-Tiering:** NOT part of Glacier — separate storage class that auto-moves objects between Standard and Standard-IA based on access patterns.
- **Glacier Deep Archive:** Cheapest S3 storage class — for data accessed once or twice a year.
- **Compliance drives Region selection** — not Governance (that's a CAF perspective). Don't let CAF vocabulary bleed into Region selection questions.
