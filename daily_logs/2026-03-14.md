## 🧠 What I Learned / Key Takeaways
- AWS Lambda functions are event-driven and receive input as JSON payloads, which the handler function processes.
- In real-world architectures, Lambda commonly responds to events from other AWS services such as S3 uploads.
- Kubernetes and container orchestration systems are divided into planes:
  - **Control plane**: manages scheduling and cluster state.
  - **Data plane**: executes workloads.
- AWS EKS manages the Kubernetes control plane so teams do not need to operate it themselves.
- AWS compute services exist on a spectrum of control:
  EC2 → Containers (ECS/EKS) → Fargate → Lambda.
