## 🧠 What I Learned / Key Takeaways
- A key is never a type — it's just a label; the type lives in the value assigned to it
- Granting a permission (IAM policy) and actually using it (a workflow step calling that action) are two separate things — neither implies the other
- Data sources in Terraform (`data` blocks) only read existing values — they never create or modify infrastructure
- CloudFront ARNs have no region segment, since CloudFront is a global service
- Non-sensitive but still-shared config values (like a CDN distribution ID) are worth centralizing once, rather than duplicating across multiple files/systems
