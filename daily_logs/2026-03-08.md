## 🧠 What I Learned / Key Takeaways
- The three AWS pricing principles: Pay as you go, Save when you commit (time-based), Pay less by using more (volume-based)
- The three primary cost drivers are compute, storage, and outbound data transfer — inbound data and same-region traffic are generally free
- AWS Budgets is proactive (alerts before overspending); AWS Cost Explorer is reactive (analyzes historical trends and forecasts); AWS Pricing Calculator is pre-deployment estimation
- AWS Organizations consolidates billing across multiple accounts and aggregates usage for volume discounts — different from the Billing Dashboard which only provides visibility
- Support plan tiers: Basic (free) → Developer (email) → Business (phone + full Trusted Advisor) → Enterprise On-Ramp (pool of TAMs, 30-min critical response) → Enterprise (designated TAM, 15-min critical response)
- Full Trusted Advisor checks unlock at Business tier and above
- AWS Marketplace is for third-party software only — native AWS services and hardware are not Marketplace products
- APN (AWS Partner Network) supports businesses that build on AWS, not end users; key benefit is Partner Funding
- Cost optimization techniques: rightsizing with AWS Compute Optimizer, Spot Instances (up to 90% off), Auto Scaling, RDS read replicas, ElastiCache caching, S3 lifecycle policies, and VPC Endpoints
- VPC Endpoints provide both a cost benefit (no public internet data transfer charges) and a security benefit (traffic stays within AWS network)
- S3 Glacier Deep Archive is the lowest-cost storage class, designed for data accessed once or twice a year
