## 🧠 What I Learned / Key Takeaways
- **ALB is the only load balancer that supports Lambda as a target** — Lambda is a Layer 7 construct, it receives HTTP requests as JSON events. NLB operates at Layer 4 and has no concept of HTTP, so it cannot forward to Lambda
- **AWS Step Functions** — serverless orchestration service that coordinates multiple AWS services into workflows using a visual state machine. Lambda executes the work, Step Functions manages the sequence. Trigger: "coordinate multiple services" + "serverless workflow"
- **Snowball Edge vs Snowcone vs Snowmobile** — Snowball Edge = physical device with onboard compute for TB-PB migrations. Snowcone = smallest, backpack-sized, remote/harsh environments. Snowmobile = semi-truck, entire data center migrations
- **S3 Lifecycle policies incur costs** — the policy itself is free but every transition it triggers generates PUT/COPY requests which are billable
- **MFA Delete** — the only mechanism that requires physical proof of identity at the moment of a destructive S3 action, regardless of IAM permissions. Trigger: unauthorized/accidental deletion
- **CloudTrail does NOT capture application error logs** — those go to CloudWatch Logs via the CloudWatch agent. CloudTrail only logs AWS API calls
- **Trusted Advisor** — analyzes your environment and provides best practice recommendations across cost optimization, performance, security, fault tolerance, and service limits. Full suite requires Business plan or above
- **Access Keys vs EC2 Key Pairs** — Access Keys authenticate programmatic AWS API/CLI calls. EC2 Key Pairs are for SSH access into EC2 instances. Completely different purposes
- **Access Keys use cryptographic signing (SigV4)** — the Secret Access Key never travels over the network. It signs the request locally and AWS verifies the signature independently. This achieves both authentication and non-repudiation
- **IAM Account Alias** — a custom name you create to substitute your 12-digit account ID in the IAM sign-in URL. You create it, AWS doesn't assign it
- **Security group rules** — source/destination must be an IP/CIDR range or a security group ID. Hostnames and instance IDs are invalid
- **EBS volume types** — throughput-intensive sequential workloads (video streaming, big data) → st1 (Throughput Optimized HDD). High IOPS transactional databases → io2. General purpose → gp3. Cold/infrequent → sc1
- **Artifact vs Audit Manager** — Artifact = download AWS's compliance reports (SOC 1, SOC 2, ISO). Audit Manager = aggregate and assess your own environment's compliance evidence
- **Reserved Instance payment options** — All Upfront (biggest discount), Partial Upfront (some now, discounted hourly rate), No Upfront (discounted hourly rate only). More upfront = bigger discount. Standard RI saves more than Convertible RI
