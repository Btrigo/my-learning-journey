## üß† What I Learned / Key Takeaways
- CloudTrail Event History provides 90 days of management event visibility by default.
- To retain logs beyond 90 days, you must create a Trail, which delivers logs to S3.
- Trails can be multi-region and (when configured properly) span AWS Organizations.
- Logging Data Events (e.g., S3 object-level activity) can become expensive quickly.
- CloudWatch Logs Insights allows interactive querying of logs without exporting to S3/Athena.
- A CloudWatch Metric is a time-ordered set of data points monitored over time.
- Many AWS services automatically publish logs or metrics to CloudWatch.
- AWS WorkDocs is similar in concept to Microsoft SharePoint (managed collaboration/document service).
- AWS Directory Service allows you to run Microsoft Active Directory as a managed AWS service.
