## 🧠 What I Learned / Key Takeaways
- **OAC (Origin Access Control):** CloudFront signs every request to S3 using SigV4. S3 bucket policy enforces that only requests from the specific CloudFront distribution ARN are allowed. Direct S3 access returns AccessDenied. OAC replaced the legacy OAI and is current AWS best practice.
- **Network devices fundamentals:** Switches connect end hosts within a LAN (typically 24+ ports). Routers connect separate networks together and route traffic over the internet. Firewalls monitor and control traffic entering/exiting a network based on configured rules. Next-generation firewalls include advanced filtering like IPS on top of traditional firewall features.
- **Client vs Server:** A client accesses a service; a server provides one. The same device can be both depending on context.
- **CCNA 200-301 exam update:** Cisco updated the exam on February 3, 2026 (still code 200-301). New topics include AI in networking, cloud network management, automation, and WPA3. Core networking content is unchanged. Next update unlikely before 2027.
